# Aula 2: Conceitos Básicos da Linguagem Move

**Vídeo**: [Assistir Online](https://drive.google.com/file/d/1bx6rKgGAX0mdgMQA002tz157mE1C66w4/view?usp=sharing)

### 1. Introdução
- **Move**: Linguagem focada em ativos digitais (resources) que não podem ser copiados ou destruídos acidentalmente.
- Criada pela Meta (ex-Libra/Diem), adaptada pela Sui com modelo de objetos.

### 2. Recursos Oficiais
- [Documentação Oficial (Inglês)](https://docs.sui.io/)
- [Repositório: Sui First Steps (Português)](https://github.com/AguaPotavel/sui-first-steps) - Guia passo a passo recomendado pelo curso.

### 3. Sui CLI
Ferramenta essencial para compilar, testar e interagir com a rede.
- **Compilar/Testar**: `sui move build`, `sui move test`
- **Interagir**: Deploy de pacotes, consultar objetos, executar transações.

### 4. Modelo de Objetos vs Contas
- **Solidity**: Dados dentro de contas.
- **Move (Sui)**: Objetos mutáveis independentes. Permite maior segurança e representação natural de ativos (jogos, DeFi).
